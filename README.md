# Data Structures & Algorithms Solutions

A curated repository containing Python solutions to various classic data structure and algorithm problems, organized systematically by topic/category to make navigation and study easy.

---

## 📂 Repository Structure

The solutions are grouped into the following categories:

```text
data_structure/
├── arrays_and_hashing/    # Array manipulation, sorting, hashing, and sliding window
├── binary_search/         # Binary search algorithms and optimization problems
├── design/                # Custom data structures and class design tasks
├── dynamic_programming/   # Dynamic programming and optimization solutions
├── graphs/                # Graph traversals (DFS/BFS) and pathfinding
├── linked_lists/          # Singly and doubly linked lists
├── math_and_bits/         # Bit manipulation and mathematical puzzles
├── strings/               # String parsing, reversals, and anagrams
└── trees/                 # Binary trees, binary search trees (BST), and traversals
```

## 🛠️ Requirements & Execution

All solutions are written in standard Python 3. To run any solution:

1. Navigate to the specific category directory:
   ```bash
   cd arrays_and_hashing
   ```
2. Run the solution using python:
   ```bash
   python3 interview01.py
   ```
